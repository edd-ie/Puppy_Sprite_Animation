Creating the play area:
    * create a <canvas> in html
    * stlye it to specific w*h and place a border color for ease
    * get into JS 
    * use getContext to allow drawing of the
    * specify drawing h & width limit.

Source the characters:
    * create and empty var the calls a new image()
    * source the sprite file
    * Set the width and height of a single sprite on the sprite sheet
    * set the frame width and height variables 

Animation/cycling the sprite sheet:
    * create an animation function 
    * clear the canvas using -> ctx.clearRect(0, 0, CANVAS_WIDTH, CANVAS_HEIGHT);
    * Draw and animate the sprite using -> ctx.drawImage(image, sx, sy, sw, sh, dx, dy, dw, dh)
        - sx  =  source x => navigate the sprite sheet in the x-axis (frameX * spriteWidth)
        - sy = source y => navigate the sprite sheet in the y-axis (frameY * spriteHeight)
        - sw = source width => select the width of the sprite image
        - sh = source height => select the height of the sprite image
        - dx = drawing x
        - dy = drawing y
        - dw = drawing width => the width of the play area
        - dh = drawing height => the height of the play area
